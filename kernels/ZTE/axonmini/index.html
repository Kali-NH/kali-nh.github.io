      По ссылке несколько архивов для установки:
      • архив с полным набором - nethunter-axonmini-lollipop-nobrand-kalifs-full-v3.15.4-170726.zip;
      • архив с минимально необходимым софтом - nethunter-axonmini-lollipop-nobrand-kalifs-minimal-v3.15.4-170726.zip;
      • архив только с ядром и модулями - kernel-nethunter-axonmini-lollipop-v3.15.4-170726.zip;
      • архив для удаления - uninstaller-nethunter-v3.15.4-170726.zip (не уверен что он работает как надо. Самый верный способ вернутся обратно - это восстановить бэкап из TWRP)
