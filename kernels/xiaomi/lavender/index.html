    • Инструкция по установке:
      Reboot into Recovery (e.g. TWRP)
      Flash official nethunter arm64 generic zip
      Flash the kernel zip attached in this post. (Reflash Magisk after that in case it didnt get preserved)
      Reboot. Install the Nethunter App and NH Terminal via Nethunter Store if they didnt get installed already.
      Give nethunter app and terminal all permissions it asks for.
      Open Nethunter app and navigate to "Kali Chroot Manager" and make sure kali chroot is getting set up and started correctly.
      Profit

      To fix camera and wifi on MIUI-ROMs, flash - MIUI.Q.OldCam+WiFi-Patch.v2.zip (8,25 мб)
      Flash this in Magisk if you want to use this kernel on a HMP ROM - SDM660_EAS_perfHAL_Module.zip (9,78 кб)
      (You need magisk for Nethunter to work. If you had it installed before and Magisk isnt present anymore, flash it again)

      Everything is fully functional, from HID to Wifi-adapter compatibility and wlan0 in monitor mode functionality.
